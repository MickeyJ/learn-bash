# Learning bash commands

<br>

## set up
make sure you can execute the __main.sh__ file
```bash
$ chmod +x ./main.sh
```

<br>

## usage
run the __main.sh__ file
```bash
$ ./main.sh hello
```

<br>

## make script global
put the program in `/usr/local/bin` to use globally
```bash
$ sudo cp ./main.sh /usr/local/bin/test-program
$ test-program hello
```
